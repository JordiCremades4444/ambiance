# Ambiance

Minimal ambient page on GitHub Pages: full-screen background, looping audio, elapsed timer. No controls — audio plays on load.

## Live

https://jordicremades4444.github.io/ambiance/

## Files

| File | Purpose |
|---|---|
| `index.html` | The page |
| `audio.mp3` | The track |
| `background.jpg` | The background |

## Swap the track or background

Replace `audio.mp3` (keep the name). Keep under ~30 min / 100 MB (GitHub limit).

Replace `background.jpg` (keep the name).
